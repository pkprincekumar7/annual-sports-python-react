## Frontend

React (Vite) frontend for the annual sports application.

### Setup

- Install dependencies: `npm install`
- Configure environment: set `VITE_API_URL` (defaults to `http://localhost:3001`; use `/` when served via nginx)
- Run locally: `npm run dev`

### Build

- Build for production: `npm run build`
- Preview production build: `npm run preview`
